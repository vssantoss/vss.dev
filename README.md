# Victor Santos

`full-stack developer` · building for the web

Hi, I'm Victor. I build clean APIs, fast interfaces, and small tools I
would actually want to use myself.

This repository powers [vss.dev](https://vss.dev), my personal site and
public workspace. The site is designed like a code editor because that is
where I spend most of my time: reading, shaping, debugging, and shipping.

## What I work on

- **Backend:** services, APIs, data modeling, and the boring parts that
  keep systems reliable.
- **Frontend:** semantic HTML, modern CSS, and JavaScript that stays close
  to the browser.
- **Product craft:** turning rough ideas into working software, then
  tightening the experience until it feels obvious.

## Current Focus

- Shipping `vss.dev` as a better home for my writing and work.
- Looking for backend or full-stack roles with teams that value shipping
  and craft.
- Building in public through notes, practical tips, and a career
  changelog.
