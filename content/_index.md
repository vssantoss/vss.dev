# Victor Santos

`full-stack developer` · building for the web

---

Hi, I'm Victor. I build software for the web: clean APIs, interfaces
that feel fast, and small tools I'd actually want to use myself.

This site is my workspace, in public. There's no landing page. You're
looking at an editor, and every section is a file you can open: notes
I'm taking, things I'm shipping, and an honest log of how my craft is
evolving.

## What I work on

- **Backend:** designing services, APIs, and the data layer underneath them
- **Frontend:** semantic HTML, modern CSS, and JavaScript without the ceremony
- **The glue:** the unglamorous but important parts like CI, deploys, and observability
